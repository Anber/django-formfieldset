Rendering With A Custom Template

You can use renderform template filter on a form or a Fieldset object to render it. By default formfieldset/form.html template will be used. You can also supply another template name as an argument to renderform like this {% templatetag openvariable %} form|renderform:"myapp/custom_form.html" {% templatetag closevariable %}.

Django-formfieldset supplies a simple formfieldset/form.html template. But this filter is most useful when you override this default template or use a custom template as a filter argument.
