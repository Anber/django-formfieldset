Rendering With Template Tags

You can use renderform template filter on a form to render it. By default your form (or fieldset) will be rendered using formfieldset/form.html template. But you can supply your custom template as an argument to renderform like this {% templatetag openvariable %} form|renderform:"myapp/custom_form.html" {% templatetag closevariable %}.
