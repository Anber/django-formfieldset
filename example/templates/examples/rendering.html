Fieldset Rendering

There are three extra methods defined on fieldset enabled forms that correspond to three rendering methods of regular forms.

django.forms.Form formfieldset.forms.FieldsetMixin
as_p as_fieldset_p
as_ul as_fieldset_ul
as_table* as_fieldset_table
*: default rendering method
{% example "rendering" %}

As you can see above rendering with fieldsets is as easy as calling {% templatetag openblock %} form.as_fieldset_p {% templatetag closeblock %}, {% templatetag openblock %} form.as_fieldset_ul {% templatetag closeblock %} or {% templatetag openblock %} form.as_fieldset_table {% templatetag closeblock %} in your template.
